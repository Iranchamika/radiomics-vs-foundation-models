# 18 — Phase 5 Statistical Synthesis Workflow

**Purpose:** Convert the locked Phase 4a/4b corpus into manuscript-ready statistical outputs — random-effects meta-analysis of Δ AUC, forest plot, three quality-appraisal heatmaps, PRISMA 2020 flow diagram, and the characteristics-of-studies supplementary table.

**Inputs (already in `.\`):**

## Methodological choices (locked)

- **Variance estimator:** Hanley-McNeil for each arm's SE(AUC). Δ AUC variance = SE_FM² + SE_Rad² (independent-arms assumption — conservative, errs on wider CIs).
- **Pooling model:** Random-effects with DerSimonian-Laird estimator (metafor `method = "DL"`).
- **Test statistic:** Knapp-Hartung adjustment for small-sample CI (metafor `test = "knha"`).
- **Heterogeneity:** I², τ², Q test, all reported.
- **Subgroup analyses (pre-specified per PROSPERO):** FM tier (DPM vs ImageNet-CNN), modality (CT / MRI / PET-CT / X-ray / Ultrasound), task type (classification / prognostic / response / multi-class).
- **Sensitivity analyses:** (a) low-ROB-only (PROBAST Overall = Unclear, n=3); (b) high-RQS (RQS ≥ median 12, n≈14); (c) externally-validated-only (n=11); (d) exclude weak-Rad-arm papers (Rad AUC < 0.60).

## Sensitivity analysis matrix

After the primary meta-analysis runs, the same script produces these alternative-specification rows for the manuscript Supplementary Material:

| Sensitivity | Inclusion criterion | Expected n | Expected pooled Δ |
|---|---|---:|---|
| Primary | All papers | 28 | +0.040 ± CI |
| Domain-pretrained FM only | FM tier = DPM | 5 | +0.140 ± CI (tight) |
| ImageNet-CNN only | FM tier = ImageNet | 23 | +0.015 ± CI (wide) |
| Externally-validated only | External_Validation = YES | 11 | +0.075 ± CI |
| Low/Unclear ROB only | PROBAST = Unclear | 3 | underpowered — narrative |
| High-RQS only | RQS ≥ 12 | ~14 | +0.060 ± CI |
| Exclude weak-Rad arm | Rad AUC ≥ 0.60 | 27 (exclude Paper 16) | +0.034 ± CI |
| AUC-only (drop C-index) | Metric = AUC | ~26 | +0.045 ± CI |

If the FM-tier subgroup estimates remain non-overlapping across sensitivity analyses (DPM CI excludes ImageNet-CNN CI), the headline finding is robust.
